## 프로젝트 소개
- 타자연습 게임을 모방하여 만든 간단한 게임입니다. 제시된 단어를 모두 입력하면 게임이 끝납니다.
<br>

<br>

## 사용 라이브러리
- React
- tailwindcss
<br>

## 핵심 기능 및 구현 방법
- 제시된 단어를 화면에 렌더링
- 입력받은 단어가 제시된 단어와 일치하면 제시된 단어가 호박 이미지로 변경됨
- 모든 단어를 입력하면 게임을 다시 하거나 끝낼 수 있음
- useContext
- spin
- useState
